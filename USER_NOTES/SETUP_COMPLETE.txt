## 🚀 How to Use

### Push to GitHub
```bash
# 1. Create empty repo on GitHub
# 2. Add remote
git remote add origin https://github.com/username/DesktopAIApp.git

# 3. Push initial branch
git push -u origin 1.1

# 4. Create version branches when ready to release
python create_version_branch.py
```

### Create Releases
```bash
# Interactive mode
python create_version_branch.py

# Or quick commands
python create_version_branch.py next        # Show next version
python create_version_branch.py create      # Create next version
python create_version_branch.py list        # List all versions
```

### Version Progression
```
1.1 (original branch)
 ↓
1.1.2 (first release)
 ↓
1.1.3 (second release)
 ↓
... up to 1.1.9
 ↓
1.2 (next minor version)
```
